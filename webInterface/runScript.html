Download from Content Directories

Check your content directory providers (Reddit, Tumblr, and Pixiv) for new content, and download it. This downloads things you have Liked, Upvoted, Hearted, or Saved (depending on your settings).

Download URLs

Paste URLs to images or videos. The script will attempt to download the content at the URL. This only works with direct links or on supported websites, e.g. YouTube.
Paste one URL per line.


Scan file system

If you want to browse content from any other source using Content Collector, you can copy the files to Content Collector's output directory.

Files with the following extensions are supported: .mp4, .webm, .gif, .jpg, .jpeg, .png, .mp4, .webm, and .riff.
